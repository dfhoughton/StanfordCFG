Download

A source tarball with documentation, ant build script, etc: StanfordCFG-__VERSION__.tgz

For the most up-to-date code, clone the github repository: 

	git clone https://github.com/dfhoughton/StanfordCFG.git

Once you have the code, most things you may want to do can be accomplished with the build file.If you wish to run the unit tests, you will need JUnit 4 on your classpath.

Note the build.properties file. You will have to modify some file paths in this for the ant build file to work.
